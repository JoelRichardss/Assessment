Typography project

Axial

An axial typographic design means that the type is on an axis.To achieve this affect, I used the rotation tool and changed the sizes of the font. I also added colours to some of the key bits of text so they would stand out. Additionally, I added green rectangles to help border the text.

Grid

For a Grid typographic design the text fits into columns or rows.So I put the text into smaller sections and aligned them into a grid shape by putting the text into columns and rows.

Bilateral

In a bilateral typographic design the type is symmetrical or centered.I divided the text into three different sections. Two smaller boxes were placed one at the top and bottom of the page, and one larger main box was positioned in the middle. I did this to make it symmetrical. Then, I made sure that the text boxes were centred.

Radial

There is a central focal point, and the text radiates out from that point.To achieve the radial effect, I added a circle to the bottom left of my canvas. Then, I positioned the text around it and changed the length of the text so it would be radiating from the circle.

Modular

Modular layouts use repeating structures to break up the content.I used a repeating structure by separating the text into boxes I made six boxes in total four of them the same size and positioned them on the left and right I also added two to the top and bottom of the page that were different sizes.

Random

In a Random typographic system the font size and placement is random.I did this by chaning the size of the font and the postion I used the type on a path tool to add a more interesting and random layout of the text I also added random sections of colour as well.

Dilatational

Dilatational designs use type set along circular paths.To make the type Dilatational I drew circles and then used the type on a path tool i used colour to highlight the key information

Transitional

The type in the Transitional typographic system moves freely, as if it's transitioning from one point to another.To do the transitional typographic system I drew lines and used the type on a path tool i made sure that the lines were transitioning from one part of the canvas to another so the type would flow.
