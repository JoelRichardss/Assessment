Typography project

Axial

An axial typographic design means that the type is on an axis.

Grid

For a Grid typographic design the text fits into columns or rows

Bilateral

In a bilateral typographic design the type is symmetrical or centered.

Radial

There is a central focal point, and the text radiates out from that point.

Modular

Modular layouts use repeating structures to break up the content

Random

In a Random typographic system the font size and placement is random

Dilatational

Dilatational designs use type set along circular paths

Transitional

The type in the Transitional typographic system moves freely, as if it's transitioning from one point to another.
